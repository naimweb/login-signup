# 🔐 Login Signup Full Stack App

This is a full-stack authentication system built with **React (frontend)** and **Node.js + Express + MongoDB (backend)**.  
It includes user registration, login, JWT authentication, and protected routes.

---

## 🚀 Features

- User Registration (Signup)
- User Login
- Password hashing using bcrypt
- JWT authentication
- Token stored in localStorage
- Frontend form validation
- API integration with Axios

---

## 🛠️ Tech Stack

### Frontend:
- React
- React Router DOM
- Axios
- Tailwind CSS

### Backend:
- Node.js
- Express.js
- MongoDB (Mongoose)
- bcryptjs
- jsonwebtoken
- dotenv
- cors
